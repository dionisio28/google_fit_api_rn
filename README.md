# React Native Project - Connect in Google Fit API


This project connects to the google fit api, using React Native.

## Prerequisites

- [Node.js > 12](https://nodejs.org) and npm (Recommended: Use [nvm](https://github.com/nvm-sh/nvm))
- [Watchman](https://facebook.github.io/watchman)
- [Xcode 12](https://developer.apple.com/xcode)
- [Cocoapods 1.10.1](https://cocoapods.org)
- [Expo](https://github.com/expo/expo-cli)
- [JDK > 11](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)
- [Android Studio and Android SDK](https://developer.android.com/studio)

## Base dependencies

- [React Native Google Fit](https://github.com/StasDoskalenko/react-native-google-fit) To communicate with the google API.
